Logical Operators

combine boolean values so that you can express more complex ideas && and, evaluates to true if the values on both are true || or, evaluates to true if either one is true ! not, works on just one value let userLoggedIn = true; let userPurchasedItem = false; // test if both values are true userLoggedIn && userPurchasedItem; // => false // test if either value is true userLoggedIn || userPurchasedItem; // => true // invert the value of a variable !userLoggedIn; // => false !userPurchasedItem; // => true short-circuiting code stops early if the first value of the || operator is truthy. returns the first truthy value, or the last value if no values are truthy code stops early if the first value of the && operator is falsy. returns the first falsy value, or the last value if no values are falsy.

Data Types

Strings - text or numbers in " " or ' ' // declare a string with quotes let city = "Seattle"; let state = "Washington"; Numbers - Integers (whole numbers) and floating point decimal numbers (2.345) Booleans - True or False // declare a boolean let isLoggedIn = true; // evaluate a boolean if (isLoggedIn) { console.log("The user is logged in."); Pronounced 'not' or 'bang', ! inverts a boolean value. This means that !false turns into true, and !true turns into false. Null - is a data type that's used to indicate that a variable has no value. null is used as the default value when other data is not available. Even though null is a distinct data type, typeof will tell you that it's an object Undefined - is a special value that means that a variable was never defined. null value means that the developer decided that the value was empty on purpose. An undefined value means that the value was never assigned. you shouldn't assign a variable undefined. If you want to represent the absence of a value in your code, use null instead let example = null; example; // null let magicWord; magicWord; // undefined

Converting Data Types

If you combine a number with a string using the + operator, the resulting value will be a string. This process of automatic conversion is called coercion. Only + parseInt() parses a string and returns an integer. But notably, parseInt returns only integers, like -5, 0, 111, 22,847, etc. It doesn't deal with decimals, like 1.618. But that's where parseFloat() comes in. toString() which turns a number into a string let age = 101; let ageString = age.toString(); typeof age; // 'number' typeof ageString; // 'string'
